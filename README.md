# Tarefas do Grupo DS-DEVS 

**Info**

Este repositório servirá para submissão dos resolução (códigos-fonte) das tarefas a serem concluidas após a atribuição individual ou em grupo.  

**Passos Para Submissão**

Passo 1: escolha a pasta da tarefa que pretende fazer a submissão da resolução

Passo 2: Clique nas reticências de seguida clique em upload files 

Passo 3: No seu armazenamento(Smart ou PC) seleccione os arquivos que deseja subir para o repositório

Passo 4: Após os arquivos terem sido carregados, seleciona o radio button Com o label "pull request" e adiciona uma breve descrição do que seu código faz e crie uma nova branch

Passo 5: click no botão commit ou pull request
